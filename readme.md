# Strawberry-Tracker
これは姿勢推定モデルをスマホでも手軽に使えるようにするプロジェクト！

# どんなことができる？
* モーションデータの作成（BVHで出力）
* モーションデータをMotionBuilderとUnityにリアルタイムで送信
* OSCでVRChatにモーションデータを送信（両足・腰・胸）
